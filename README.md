# dotfiles
my i3-gaps theme

## stuff to install
* rofi
* i3-lock-fancy/lock
* polybar
* termite
* playerctl
* pulseaudio
* feh
* scrot
* lxappearance
* compton
* alsa-utils
* nerd-fonts-complete
* ttf-font-awesome

## installation
copy the files in your home directory in the corresponding folders.

### lxappearance
I have got the numix theme with the paper icon theme and the paper mouse cursor. and the SFNS Display font at a fontsize of 13.
